** Clustering **

This analysis was to segment customer profiles based on their demographic info, lifestyle and behaviour. The groups were made for a new product launch. Clustering was done to see that which customer groups will be best fit for this new product so that they can be targeted first during marketing.


Documents attached in this folder:

1. Input data files (pda_2001.csv & pda_disc2001.csv)
2. SAS code to run the model
3. Output word file


How to run the code?

Step 1. Please download the (pda_2001.csv & pda_disc2001.csv files in your local system.
Step 2. Note the file path where you have saved the file.
Step 3. Set the 'libname' as the name of your drive/folder where the files are saved(libname hw4 "F:/CustomerProfile_Segmentation").Mention the path in the beginning of code where 'proc import datafile' is written. It's the fourth and tenth line of code.
Step 4. Please change the "\" to "/" in 'libname hw4 "F:/CustomerProfile_Segmentation"' line otherwise the code will throw error.
Step 5. Yes, now you are ready to run the code.
Step 6. After the code is run successfuly there be an output file generated in SAS Framework itself.
